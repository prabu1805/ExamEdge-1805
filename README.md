🚀 ExamEdge – Online Mock Test Platform

ExamEdge is a simple and interactive online mock test platform built using HTML, CSS, and JavaScript.
It allows users to sign up, log in, attempt timed quizzes, navigate questions easily, and view leaderboard rankings.

This project simulates a real online exam environment similar to competitive exam platforms.

📌 Features

✨ User Authentication

Signup and Login system

Stores user data using browser LocalStorage

⏱ Exam Timer

Countdown timer for the test

Automatically ends when time expires

📝 Question Navigation

Question palette to jump between questions

Previous and Next buttons

✔ Answer Selection

Multiple choice questions

Selected answers highlighted

🔖 Mark for Review

Users can mark questions for review and revisit later

📊 Leaderboard

Displays top scores

Ranks users based on performance

🎨 Modern UI

Gradient backgrounds

Responsive layout

Clean and interactive design

🛠️ Technologies Used


HTML	Structure of the application

CSS	Styling and layout

JavaScript	Functionality and logic

LocalStorage	Data storage for users and scores

🎮 How It Works

1️⃣ User creates an account using Signup page

2️⃣ User logs in using Login page

3️⃣ User starts the mock test

4️⃣ Questions can be:

Answered

Skipped

Marked for review

5️⃣ After completing the exam:

Score is calculated

Leaderboard is updated
